## Make a .env file in the exact same folder where main.py is located and copy paste this:

```
COHERE_API_KEY="YourCohereAPIKey"
TAVILY_API_KEY="YourTavilyAPIKey"
```

- replace "YourCohereAPIKey" with your generated Cohere API key and
- replace "YourTavilyAPIKey" with your generated Tavily Search API key
